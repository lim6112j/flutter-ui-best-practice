# gecko_app

A new Flutter project.

## Getting Started

* open simulator
`open -a Simulator`
* run 
`flutter run`
## how to add column
* add column to dbhelper.dart
* update version number
* change gecko.dart model
* delete app from ios simulator 
* flutter run stop and restart
